# Project Description: git-practice-Md.Masrul-Sakib

## Author Information
- **Name:** Md. Masrul Sakib

## Project Objective
The primary goal of this project is to practice and demonstrate proficiency in basic Git and GitHub workflows. This includes:
- Repository initialization and cloning.
- Branching and merging.
- Committing changes with clear messages.
- Pushing/Pulling from remote repositories.
- Managing documentation (README and Project Description).

## Technical Overview
This project contains a simple Python application consisting of two main files:

### 1. `src/main.py`
The entry point of the application. It performs the following actions:
- Imports the `date` class from the `datetime` module to display the current date.
- Imports helper functions from `utils.py`.
- Prints a personalized greeting with the author's name.
- Displays the current date.
- Executes and prints results from basic mathematical functions (addition and subtraction).

### 2. `src/utils.py`
A utility module containing reusable functions:
- `add(a, b)`: Returns the sum of two numbers.
- `subtract(a, b)`: Returns the difference between two numbers.

##  Environment Requirements
- **Language:** Python 3.x
- **Standard Libraries:** `datetime`

## Summary
This project serves as a foundational exercise for anyone starting with Python and Git. It emphasizes clean code structure, modularity (using `utils.py`), and the importance of well-maintained documentation.
